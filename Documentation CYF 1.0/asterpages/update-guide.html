Update Migration Guide



This page is the guide that updating lua script to new AsteriskMod's version.


Common



info.cyfmod


At first, change target-version parameter to AsteriskMod's latest version
Latest version is v0.5.3, so you write like below: 

-- info.cyfmod:
target-version="v0.5.3"



v0.5.2.x -> v0.5.3



It is not too much to say that this version changes everything about past version features. That's because v0.5.3 contains clean-up of C# and Lua code.

The PlayerUtil Object


The functions of moving all UI's parent object is renamed:

  • PlayerUtil.UIMove()  ->  PlayerUtil.Move()
  • PlayerUtil.UIMoveTo()  ->  PlayerUtil.MoveTo()
Below variables getting position is added instead of PlayerUtil.GetUIPosition():
  • PlayerUtil.x
  • PlayerUtil.y
From 
PlayerUtil.UIMove(10, -3)
PlayerUtil.UIMoveTo(-9, 73)
local pos = PlayerUtil.GetUIPosition()
if pos[1] == 10 or pos[2] == -5 then
end
To 
PlayerUtil.Move(10, -3)
PlayerUtil.MoveTo(-9, 73) -- or  PlayerUtil.x = -9  PlayerUtil.y = 73
if PlayerUtil.x == 10 or PlayerUtil.y == -5 then
end


The functions of moving the texts showing name and love is renamed:
  • PlayerUtil.NameUIMove()  ->  PlayerUtil.Name.Move() and PlayerUtil.Love.Move()
  • PlayerUtil.NameUIMoveTo()  ->  PlayerUtil.Name.MoveTo() and PlayerUtil.Love.MoveTo()
Below variables getting position is added instead of PlayerUtil.GetNameUIPosition():
  • PlayerUtil.Name.x
  • PlayerUtil.Name.y
  • PlayerUtil.Love.x
  • PlayerUtil.Love.y
The text showing the player's name and The text showing the player's love are separated on AsteriskMod v0.5.3, so you can modify each text.
From 
PlayerUtil.NameUIMove(-25, -41)

PlayerUtil.NameUIMoveTo(73, 15)

local pos = PlayerUtil.GetNameUIPosition()
if pos[1] == -42 or pos[2] == 97 then
end
To 
PlayerUtil.Name.Move(-25, -41)
PlayerUtil.Love.Move(-25, -41)

PlayerUtil.Name.MoveTo(73, 15) -- or  PlayerUtil.Name.x = 73  PlayerUtil.Name.y = 15
PlayerUtil.Love.MoveTo(73, 15) -- or  PlayerUtil.Love.x = 73  PlayerUtil.Love.y = 15

if PlayerUtil.Name.x == -42 or PlayerUtil.Name.y == 97 then
end
-- or
-- if PlayerUtil.Love.x == -42 or PlayerUtil.Love.y == 97 then
-- end


Use variable of static text if the texts showing name and love wanna be colored:
  • PlayerUtil.SetNameColor()  ->  PlayerUtil.Name.color
  • PlayerUtil.SetNameColor32()  ->  PlayerUtil.Name.color32
  • PlayerUtil.SetLVColor()  ->  PlayerUtil.Love.color
  • PlayerUtil.SetLVColor32()  ->  PlayerUtil.Love.color32
If you wanna change that alpha, use StaticText.alpha or StaticText.alpha32.
From 
PlayerUtil.SetNameColor(1, 0, 0)
PlayerUtil.SetNameColor32(255, 166, 0)
PlayerUtil.SetLVColor(1, 1, 0)
PlayerUtil.SetLVColor32(0, 192, 0)
To 
PlayerUtil.Name.color = {1, 0, 0}
PlayerUtil.Name.color32 = {255, 166, 0}
PlayerUtil.Love.color = {1, 1, 0}
PlayerUtil.Love.color32 = {0, 192, 0}
