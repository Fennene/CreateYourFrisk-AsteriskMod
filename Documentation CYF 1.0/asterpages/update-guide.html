Update Migration Guide



This page is the guide that updating lua script to new AsteriskMod's version.


Common



info.cyfmod


At first, change target-version parameter to AsteriskMod's latest version
Latest version is v0.5.3.0.4, so you write like below: 

-- info.cyfmod:
target-version="v0.5.3.0.4"



v0.5.2.x -> v0.5.3



It is not too much to say that this version changes everything about past version features. That's because v0.5.3 contains clean-up of C# and Lua code.

Special Variables


The variables that changes the color of bars showing enemy's HP are renamed:

  • minibgcolor  ->  menubgbarcolor
  • bgcolor  ->  bgbarcolor
  • fillcolor  ->  menufillbarcolor and fillbarcolor
Since v0.5.3, you can set each color of each bar by using menufillbarcolor and fillbarcolor.
From 
minibgcolor = {127, 127, 127}

bgcolor = {255, 0, 0}

fillcolor = {0, 192, 0}
To 
menubgbarcolor = {127, 127, 127}

bgbarcolor = {255, 0, 0}

menufillbarcolor = {0, 192, 0}
fillbarcolor = {0, 192, 0}



Game Events


OnHPChanged() is removed.
The text showing the Player's HP does not reset color when the text is changed.



The Text Object


Below functions are no longer experimental features:

  • Text.SetSoundVolume()
  • Text.GetSoundVolume()
The functions are deleted and use below instead of those:
  • Text.SetSoundMute()  ->  Text.SetSoundVolume()
  • Text.GetSoundMute()  ->  Text.GetSoundVolume()
From 
text.SetSoundMute(true)

if text.GetSoundMute() then
end
To 
text.SetSoundVolume(0)

if text.GetSoundVolume() == 0 then
end



The PlayerUtil Object


The functions of moving all UI's parent object are renamed:

  • PlayerUtil.UIMove()  ->  PlayerUtil.Move()
  • PlayerUtil.UIMoveTo()  ->  PlayerUtil.MoveTo()
Below variables getting position is added instead of PlayerUtil.GetUIPosition():
  • PlayerUtil.x
  • PlayerUtil.y
From 
PlayerUtil.UIMove(10, -3)
PlayerUtil.UIMoveTo(-9, 73)
local pos = PlayerUtil.GetUIPosition()
if pos[1] == 10 or pos[2] == -5 then
end
To 
PlayerUtil.Move(10, -3)
PlayerUtil.MoveTo(-9, 73) -- or  PlayerUtil.x = -9  PlayerUtil.y = 73
if PlayerUtil.x == 10 or PlayerUtil.y == -5 then
end


Below functions have similar features of below functions:
  • PlayerUtil.NameUIMove()  ->  PlayerUtil.Name.Move() and PlayerUtil.Love.Move()
  • PlayerUtil.NameUIMoveTo()  ->  PlayerUtil.Name.MoveTo() and PlayerUtil.Love.MoveTo()
The text showing the player's name and The text showing the player's love are separated on AsteriskMod v0.5.3, so you can modify each text.
Note that Static Text's position is not relative position to initial position!


Use variable of static text if the texts showing name and love wanna be colored:
  • PlayerUtil.SetNameColor()  ->  PlayerUtil.Name.color
  • PlayerUtil.SetNameColor32()  ->  PlayerUtil.Name.color32
  • PlayerUtil.SetLVColor()  ->  PlayerUtil.Love.color
  • PlayerUtil.SetLVColor32()  ->  PlayerUtil.Love.color32
If you wanna change that alpha, use StaticText.alpha or StaticText.alpha32.
From 
PlayerUtil.SetNameColor(1, 0, 0)
PlayerUtil.SetNameColor32(255, 166, 0)
PlayerUtil.SetLVColor(1, 1, 0)
PlayerUtil.SetLVColor32(0, 192, 0)
To 
PlayerUtil.Name.color = {1, 0, 0}
PlayerUtil.Name.color32 = {255, 166, 0}
PlayerUtil.Love.color = {1, 1, 0}
PlayerUtil.Love.color32 = {0, 192, 0}


The functions of moving HP's UI's parent object are renamed:
  • PlayerUtil.HPUIMove()  ->  PlayerUtil.HPMove()
  • PlayerUtil.HPUIMoveTo()  ->  PlayerUtil.HPMoveTo()
Below variables getting position are added instead of PlayerUtil.GetHPUIPosition():
  • PlayerUtil.hpx
  • PlayerUtil.hpy
From 
PlayerUtil.HPUIMove(-22, 48)
PlayerUtil.HPUIMoveTo(14, -17)
local pos = PlayerUtil.GetHPUIPosition()
if pos[1] == 27 or pos[2] == 37 then
end
To 
PlayerUtil.HPMove(-22, 48)
PlayerUtil.HPMoveTo(14, -17) -- or  PlayerUtil.hpx = 14  PlayerUtil.hpy = -17
if PlayerUtil.hpx == 27 or PlayerUtil.hpy == 37 then
end


Use variable of sprite object if the labels written "HP" wanna be colored:
  • PlayerUtil.SetHPLabelColor()  ->  PlayerUtil.HPLabel.color
  • PlayerUtil.SetHPLabelColor32()  ->  PlayerUtil.HPLabel.color32
If you wanna change that alpha, use sprite.alpha or sprite.alpha32.
From 
PlayerUtil.SetHPLabelColor(0.26, 0.89, 1)
PlayerUtil.SetNameColor32(0, 60, 255)
To 
PlayerUtil.HPLabel.color = {0.26, 0.89, 1}
PlayerUtil.HPLabel.color32 = {0, 60, 255}


Use variable of life bar if the length of the bar showing the Player's HP wanna be changed:
  • PlayerUtil.SetHPBarLength()  ->  PlayerUtil.HPBar.maxhp
From 
PlayerUtil.SetHPBarLength(85)
To 
PlayerUtil.HPBar.maxhp = 85


Use variable of life bar if the bar showing the Player's HP wanna be colored:
  • PlayerUtil.SetHPBarBGColor()  ->  PlayerUtil.HPBar.bgcolor
  • PlayerUtil.SetHPBarBGColor32()  ->  PlayerUtil.HPBar.bgcolor32
  • PlayerUtil.SetHPBarFillColor()  ->  PlayerUtil.HPBar.fillcolor
  • PlayerUtil.SetHPBarFillColor32()  ->  PlayerUtil.HPBar.fillcolor32
If you wanna change that alpha, use LifeBar.bgalpha, LifeBar.bgalpha32, LifeBar.fillalpha or LifeBar.fillalpha32.
From 
PlayerUtil.SetHPBarBGColor(0.84, 0.21, 0.85)
PlayerUtil.SetHPBarBGColor32(0, 162, 232)
PlayerUtil.SetHPBarFillColor(0, 0, 0)
PlayerUtil.SetHPBarFillColor32(127, 127, 127)
To 
PlayerUtil.HPBar.bgcolor = {0.84, 0.21, 0.85}
PlayerUtil.HPBar.bgcolor32 = {0, 162, 232}
PlayerUtil.HPBar.fillcolor = {0, 0, 0}
PlayerUtil.HPBar.fillcolor32 = {127, 127, 127}


Below functions have similar features of below functions:
  • PlayerUtil.HPTextMove()  ->  PlayerUtil.HPText.Move()
  • PlayerUtil.HPTextMoveTo()  ->  PlayerUtil.HPText.MoveTo()
Note that Static Text's position is not relative position to initial position!


Use the function of static text if the text showing the Player's HP wanna be changed:
  • PlayerUtil.SetHPText()  ->  PlayerUtil.HPText.SetText()
From 
PlayerUtil.SetHPText("1 / 1")
To 
PlayerUtil.HPText.SetText("1 / 1")


Use variables of static text if the text showing the Player's HP wanna be colored:
  • PlayerUtil.SetHPTextColor()  ->  PlayerUtil.HPText.color
  • PlayerUtil.SetHPTextColor32()  ->  PlayerUtil.HPText.color32
If you wanna change that alpha, use StaticText.alpha or StaticText.alpha32.
From 
PlayerUtil.SetHPTextColor(1, 1, 1)
PlayerUtil.SetHPTextColor32(192, 192, 192)
To 
PlayerUtil.HPText.color = {1, 1, 1}
PlayerUtil.HPText.color32 = {192, 192, 192}


Below the type of function's argument is changed:
  • PlayerUtil.CreateLifeBar()  (  ->  )
Since v0.5.3, sprite layer can be used on The LifeBar Object. See The LifeBar Object for more info.
From 
bar1 = PlayerUtil.CreateLifeBar()
bar2 = PlayerUtil.CreateLifeBar(false)
bar3 = PlayerUtil.CreateLifeBar(true)
To 
bar1 = PlayerUtil.CreateLifeBar()
bar2 = PlayerUtil.CreateLifeBar("AboveHPBar")
bar3 = PlayerUtil.CreateLifeBar("BelowHPBar")



The LifeBar Object


Below variables are renamed:

  • lifeBar.value  ->  lifeBar.hp
  • lifeBar.length  ->  lifeBar.maxhp
From 
bar.length = 35
bar.value = 24
To 
bar.maxhp = 35
bar.hp = 24


Use below parameters instead of lifeBar.SetValue() and lifeBar.SetLength().
  • lifeBar.limited
  • lifeBar.hp
  • lifeBar.maxhp
  • lifeBar.SetHP()


Use variable of life bar if the bars wanna be colored:
  • lifeBar.SetBackgroundColor()  ->  lifeBar.bgcolor
  • lifeBar.SetBackgroundColor32()  ->  lifeBar.bgcolor32
  • lifeBar.SetFillColor()  ->  lifeBar.fillcolor
  • lifeBar.SetFillColor32()  ->  lifeBar.fillcolor32
If you wanna change that alpha, use lifeBar.bgalpha, lifeBar.bgalpha32, lifeBar.fillalpha or lifeBar.fillalpha32.
From 
lifeBar.SetBackgroundColor(0.7, 0, 0)
lifeBar.SetBackgroundColor32(255, 64, 64)
lifeBar.SetFillColor(0, 0.5, 1)
lifeBar.SetFillColor32(0, 64, 255)
To 
lifeBar.bgcolor = {0.7, 0, 0}
lifeBar.bgcolor32 = {255, 64, 64}
lifeBar.fillcolor = {0, 0.5, 1}
lifeBar.fillcolor32 = {0, 64, 255}



The ButtonUtil Object


Below function is renamed:

  • ButtonUtil.RevertAll()  ->  ButtonUtil.Revert()
From 
ButtonUtil.RevertAll()
To 
ButtonUtil.Revert(true, false)



The Button Object


The feature of the below function is changed into the variables:

  • button.GetActive()  ->  button.isactive
  • button.SetColor()  ->  button.color (and button.alpha)
  • button.SetColor32()  ->  button.color32 (and button.alpha32)
From 
if not button.GetActive() then return end
button.SetColor(0.5, 0.5, 0)
button.SetColor32(252, 127, 0)
To 
if not button.isactive then return end
button.color = {0.5, 0.5, 0}
button.color32 = {252, 127, 0}


button.SetSprite()'s 3rd argument is changed:
  • ""  ->  "UI/Buttons"
button.SetSprite()'s 4th argument is added and you should assign true.
From 
button.SetSprite("UI/Buttons/fightbt_0", "UI/Buttons/fightbt_1")
button.SetSprite("actbt_0", "actbt_1", "UI/Buttons")
button.SetSprite("UI/AlternativeButtons/itembt_0", "UI/AlternativeButtons/itembt_1")
button.SetSprite("mercybt_0", "mercybt_1", "UI/AlternativeButtons")
To 
button.SetSprite("UI/Buttons/fightbt_0", "UI/Buttons/fightbt_1", "")
button.SetSprite("actbt_0", "actbt_1")
button.SetSprite("UI/AlternativeButtons/itembt_0", "UI/AlternativeButtons/itembt_1", "", true)
button.SetSprite("mercybt_0", "mercybt_1", "UI/AlternativeButtons", true)


The below function have similar features to below function:
  • button.ResetColor()  ->  button.color = {1, 1, 1} or button.color32 = {255, 255, 255}
  • button.SetSize()  ->  button.Scale()
  • button.ResetSize()  ->  button.Scale()


Below function is no longer experimental features:
  • button.Scale()



The ArenaUtil Object


Below function is renamed:

  • ArenaUtil.SetDialogTextPosition()  ->  ArenaUtil.DialogTextMoveTo()
From 
ArenaUtil.SetDialogTextPosition(0, -6)
To 
ArenaUtil.DialogTextMoveTo(0, -6) -- or  ArenaUtil.dialogtexty = -6


The feature of the below function is changed into the variables:
  • ArenaUtil.SetAsteriskChar()  ->  ArenaUtil.AsteriskChar
  • ArenaUtil.SetStarChar()  ->  ArenaUtil.AsteriskChar
From 
ArenaUtil.SetAsteriskChar('＊')
To 
ArenaUtil.AsteriskChar = '＊'


Below functions are no longer experimental features:

  • ArenaUtil.SetDialogTextVolume()
  • ArenaUtil.GetDialogTextVolume()
The functions are deleted and use below instead of those:
  • ArenaUtil.SetDialogTextMute()  ->  ArenaUtil.SetDialogTextVolume()
  • ArenaUtil.GetDialogTextMute()  ->  ArenaUtil.GetDialogTextVolume()
From 
ArenaUtil.SetDialogTextMute(true)

if ArenaUtil.GetDialogTextMute() then
end
To 
ArenaUtil.SetDialogTextVolume(0)

if ArenaUtil.GetDialogTextVolume() == 0 then
end
