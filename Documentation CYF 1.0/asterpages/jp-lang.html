.langファイル & Langオブジェクト



英語以外の言語を簡単に追加、翻訳しやすいように作られたシステムです。

.langファイル



全てのlangファイルは[Modのルートフォルダ]/Localizationフォルダ直下に入っている必要があります。
(拡張子は .langです。)

英語用のファイル名は en.lang、 日本語用のファイル名は ja.lang と決まっています。


langファイルの書式は iniファイルと似ています。


パラメータ
重要な要素が「パラメータ」で、 個々のパラメータは「キー」と「値」を持ち、等号（=）で区切られます。 等号の左辺がキー、右辺が値です。
値は 文字列型（） と 文字列配列型（） の二種類があります。
キーは 半角英数字 及び 半角ハイフン（-）、 半角アンダーバー（_）のみを対応します。
また、文字列は 半角ダブルクォーテーション（"）で、 文字列配列は 半角波括弧（{}）で 囲まれている必要があります。 

String="value"
stringArray={"value1","value2"}

セクション
パラメータは 任意の名前を持つ「セクション」でグループ分けすることができます。 セクション名は角カッコ ([XXX]) で囲み1行を占めます。
セクションの宣言の後に現れるパラメータは、そのセクションに属するようになります。
明示的にセクションの終わりを示す区切り文字はなく、 次のセクションの宣言かファイルの終端でセクションは終わるため、セクションをネストすることはできません。
langファイルはiniファイルとは異なり、 どのセクションにも属さないパラメータを含むことができます。
セクション名は 半角英数字 のみを対応します。 
[section]

コメント
行の先頭にセミコロン（;）を２つ置くか、 セミコロンと半角スペースを置くことによって、 その行はコメントになります。 
;;comment
; also comment

エスケープシーケンス
エスケープシーケンスは、改行（\n）と 復帰（\r）及び \記号（\\）を対応します。


いずれも書式を間違えた場合、その行は無視されます。 （ただし、セクションの書式を間違えた場合は、次のセクションまでまるごと無視されます。）


The Lang Object [E/M/W/S/GS]


名前空間: AsteriskMod
コード: AsteriskMod/Localization/Lang.cs


Localizationフォルダ直下にあるlangファイルは、 エンジンが自動的に読み込み、このLangオブジェクトで使用できます。



Lang.SetTargetLanguage( langName) 読み取るlangファイルを指定します。
基本的には、拡張子を除くlangファイル名か、言語名を指定します。
存在しない場合は、自動的に英語（en.lang）が指定されます。

デフォルトでは自動的にユーザー（プレイヤー）の設定した言語に基づいて設定されます。
（設定で 英語なら en.langが、 日本語なら ja.langが 指定される。）


Lang.Get( key)
 or Lang[ key]
 どのセクションにも属さないパラメータを、 文字列型として取得します。
存在しない場合は空の文字列を返します。


Lang.Get( sectionName, key) セクション名とキーを指定して、 パラメータを文字列型として取得します。 存在しない場合は空の文字列を返します。


Lang.GetRaw( key) どのセクションにも属さないパラメータを、 文字列配列型として取得します。 存在しない場合は空の配列を返します。


Lang.GetRaw( sectionName, key) セクション名とキーを指定して、 パラメータを文字列配列型として取得します。 存在しない場合は空の配列を返します。


Lang.Set( key, value) どのセクションにも属さないパラメータを、 文字列型としてとして設定します。
存在しないキーの場合は自動的に追加されます。
実際のファイルの内容は変更されないことに注意してください。


Lang.Set( sectionName, key, value) セクション名とキーを指定して、 パラメータを文字列型としてとして設定します。
存在しないセクションの場合は自動的に追加されます。
存在しないキーの場合は自動的に追加されます。
実際のファイルの内容は変更されないことに注意してください。


Lang.SetRaw( key, values) どのセクションにも属さないパラメータを、 文字列配列型としてとして設定します。
存在しないキーの場合は自動的に追加されます。
実際のファイルの内容は変更されないことに注意してください。


Lang.SetRaw( sectionName, key, values) セクション名とキーを指定して、 パラメータを文字列配列型としてとして設定します。
存在しないセクションの場合は自動的に追加されます。
存在しないキーの場合は自動的に追加されます。
実際のファイルの内容は変更されないことに注意してください。


例



Modの階層は次のようなものとする。 

LangObjectExampleMod
├─ Localization
│  ├─ en.lang
│  └─ ja.lang
├─ Lua
│  ├─ Encounters
│  │  └─ encounter.lua
│  ├─ Monsters
│  │  └─ poseur.lua
│  └─ Waves
└─ Sprites

Localization/en.lang 
PlayerName="Nil256"
MonsterName="Poseur"
[EncounterTexts]
FirstTurn="Poseur strikes a pose!"

Localization/ja.lang 
PlayerName="にるにころ"
MonsterName="ポーザー"
[EncounterTexts]
FirstTurn="ポーザーが  ポーズを  くりだしてきた！"

Lua/Encounters/encounter.lua 
function EncounterStarting()
    JPStyle.SetActive(Language == "Japanese")
    Player.name = Lang.Get("PlayerName")
    -- または Player.name = Lang["PlayerName"]
    encountertext = Lang.Get("EncounterTexts", "FirstTurn")
    -- encountertext = Lang["EncounterTexts"]["FirstTurn"]とはできない。
end

Lua/Monsters/poseur.lua 
name = Lang.Get("MonsterName")
-- または name = Lang["MonsterName"]
