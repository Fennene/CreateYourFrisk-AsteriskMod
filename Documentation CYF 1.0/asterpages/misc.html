Options



Asterisk Mod also adds some options.
You can see that info and options when you select Asterisk Mod Option.
There are below options available:

  • Show Always The Description: Switch visibility of mods' description.

  • Show Error Dog: Switch visibility of the dog in error screen.

  • Experimental Features: Allow Experimental Features.
    Experiment features contains the contents which will be added future update and the contents that may occur some error or may be unstable.

  • Add option "Active" that can be disabled AsteriskMod.
  • Add option "Crate Your Frisk" that can be control CRATE YOUR FRISK.
  • Add the variable "null" that is null.
  • Add the variable "None" that is null.
  • Add the variable "UnsupportedVersion".
  • Add the function "IsThisReallyCreateYourFriskAsteriskModAprilVersion()" that can be checked about whether users use v0.5.2.9.AprilFool.
  • Add the function "GetMayBeTrue()" that returns maybe true.
  • StringStartsWith(string, string) return boolean
  • StringEndsWith(string, string) return boolean
  • StringContains(string, string) return boolean
  • StringSplit(string, string) return string[]




本来、AsteriskModはCYFのコード書き換えのみで使用できるMod、
すなわち、"Create Your Frisk 0.6.5_Data/Managed/Assembly-CSharp.dll"の置き換えのみで 使用できるように目指していました。
後に、CYFのライセンスから上記の方法を取るより、CYFエンジン自体(Unityプロジェクト)を変えて、 Githubにあげる方がより良い選択だと思い、この形で公開しています。

コードの書き換えのみ、つまり、Unityでオブジェクトを追加しないで作るつもりでした。
現在のコードも、なるべくそのつもりで書いています(が、そのせいでスパゲッティコードになっていますね...)。
ただ、これを言い換えれば、エイプリルフールとして追加した"Active"(AsteriskModの機能のON/OFF)がそこまで苦労せずに作れちゃいます。

AsteriskModで絶対に追加しようと思っている機能は、１つ、"CustomState"、そしてもう１つが"GameObject Modifying System (GMS)"です。
GMSはv0.5.3で追加しようと考えています。
その前にコードを整理する必要がありそうですが...

それとも１つ、ちょっとしたものを追加しようかと思っているので
これからもよろしくお願いします。
