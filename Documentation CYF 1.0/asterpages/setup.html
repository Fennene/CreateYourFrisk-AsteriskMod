Your Mod's Information



Asterisk Mod adds the way to add a description of your own mod.

You can see the description by pressing V key.


If you add the description of your mod, you should do that:

  1. prepare a text file written the description.
  2. Put it somewhere in your own mod.
  3. Create a file named info.cyfmod directly under the directory of your mod.
  4. Write descfile="[the path of the text file]" to info.cyfmod

Note that the engine will NOT read the files (info.cyfmod and specified files in it) are over 1MB.


For example:

-- files:
YOURMOD
├─ Audio
├─ Lua
├─ Sounds
├─ Sprites
├─ description.txt -- <= 1MB
└─ info.cyfmod -- <= 1MB

-- info.cyfmod:
descfile="description.txt"



info.cyfmod is written like .ini file. 

key="parameter"
There are available keys in info.cyfmod:

  • align - sets alignment of the description. UpperLeft by default.
    You can choose the paramter in the following:
    • UpperLeft
    • UpperCenter
    • UpperRight
    • MiddleLeft
    • MiddleCenter
    • MiddleRight
    • LowerLeft
    • LowerCenter
    • LowerRight

  • descfile - sets the path of the file written the description.

Asterisk Mod adds and modifies below features to original CYF:

  • PlayerUtil: controls the UI of the player's status.

  • ButtonUtil: controls the Buttons.

  • CustomState: You can create your own state.

  • Special Variables: control the UI for example, text of MERCYMENU and enemy's life bar.

  • Game Events: adds functions which run some the time point.

  • The Misc Object: reverses and rotates screen.

  • are only usable in AsteriskMod.

  • are only usable in AsteriskMod when "Experimental Features" is enabled.

  • Indicates content new or changed CYF's content in Asterisk Mod.


Asterisk Mod also adds some options.
You can see that info and options when you select Asterisk Mod Option.
There are below options available:

  • Experimental Features: Allow experimental features.
    Experiment features contains the contents which will be added future update and the contents that may occur some error or may be unstable.
