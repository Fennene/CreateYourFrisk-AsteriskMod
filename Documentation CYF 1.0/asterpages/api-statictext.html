The StaticText object [E/M/W]



This is derivation of The Text Object, it is made for running CYF faster.
This text object is used for the texts unneeded any update features.
That means, the text is showed instantly, there are no text effect, command, bubble and voice.
This is used for the text showing Player's status in the engine by default.


Huh? Is it very useless?
If you think like that, you don't need to see this page.
This is used in The PlayerUtil Object, but this is made based on normal Text Object.
That means, the functions' names are same to normal Text Object, so you can call functions like calling functions as Text Object.
So... what you need to see is this.



CreateStaticText( font = "uidialog", text, , position, textWidth, layer = "BelowPlayer", charspacing = nil, linespacing = 0) returns StaticText [E/M/W/S] This function creates a StaticText Object and returns it. However, as you can see, a lot of parameters are needed. Here is what you need to set:

  • font - The font of text. It's the same as text.SetFont().
    Returns an error if the font doesn't exist.

  • text - The text to display. Should just be a single string, you can not assign table of string.
    Note that StaticText can NOT use Text Commands.

  • position - Two numbers: The x and y positions of the center of the object. Sorry if it may be hard to place it, but you'll be able to use MoveTo to adjust it afterwards. You only need two numbers, like this: {320, 240}

  • textMaxWidth - You'll always need this!
    Example: 150

  • layer - The sprite layer of the StaticText Object. If it doesn't exist, it returns an error. This argument is optional. If it's not provided, the StaticText Object will be in the layer BelowPlayer (or Default in the Overworld).
    Example: "BelowPlayer"

  • charspacing - nil or .
    This is same as the text command [charspacing].

  • linespacing - This is same as the text command [linespacing].

There is NOT a one-frame delay in the StaticText Object.


AutoRefreshStaticText( active) [E/M/W/S] If true, the engine will call StaticText.SetText() automatically after you call StaticText.SetFont(), StaticText.SetCharSpacing() or StaticText.SetLineSpacing().
Default by false.


UIStaticTextObject Only


This is only for PlayerUtil's StaticText object.


UIStaticText.SetControlOverride( active) If true, the text displaying the player's status does not update anymore.


Note that static text obejct of UI can not be called StaticText.DestroyText().


Common Variables and Functions



StaticText.Remove()
 orStaticText.DestroyText() This function destroys a static text object, similar to .Remove() for bullets and sprites.

Trying to get, set or call almost anything besides StaticText.isactive on a destroyed static text object will error.


StaticText.isactive (Read-Only) Tells you if a static text object is active.

A static text object will become inactive if you call StaticText.DestroyText().


StaticText.x The local x position of the object, measured from the position of the static text object's parent. This value depends on the object's pivot.


StaticText.y The local y position of the object, measured from the position of the static text object's parent. This value depends on the object's pivot.


StaticText.absx The absolute x position of the object, measured from the bottom left corner of the screen. This value depends on the object's pivot.


StaticText.absy The absolute y position of the object, measured from the bottom left corner of the screen. This value depends on the object's pivot.


StaticText.textMaxWidth Get or set the maximum width of the text. If the value is lower than 16, it'll be set back to 16.


StaticText.layer The sprite layer of the StaticText Object. If it doesn't exist, it returns an error. This is "BelowPlayer" by default.


, , , = 1 StaticText.color Get or set the color of the text, as a table of 3 or 4 values from 0 to 1.
For example, statictext.color = {1.0, 0.0, 0.0} makes the text red.
Black areas are not affected by coloration.
The 4th value is the alpha (transparency) value of the text.


, , , = 255 StaticText.color32 Get or set the color of the text, as a table of 3 or 4 integer values from 0 to 255.
For example, statictext.color = {255, 0, 0} makes it red.
Black areas are not affected by coloration.
The 4th value is the alpha (transparency) value of the text.


StaticText.alpha Get or set the text's transparency, as a value between 0 and 1.


StaticText.alpha32 Get or set the text's transparency, as an integer between 0 and 255.


sprite StaticText.GetLetters() Returns a table containing sprite objects representing letters to be displayed.

Letter sprite objects can not have their layer changed, can not run sprite.Mask, and can only be parented to other letter sprites.

NOTE: Accessing this table generates the table anew every time. To save on resources, store this table to a local variable before doing any operations on it. Even code such as for i = 1, #statictext.GetLetters() do is a bad idea.


StaticText.SetParent(sprite parent) Sets the text object's parent.
The static text object will follow around the parent object as it moves.


StaticText.SetAnchor( x, y) If this static text object has a parent set via StaticText.SetParent, this will control its anchor point (the relative point on the parent sprite that the text object will follow if the parent sprite gets scaled).

x and y should usually both be between 0 and 1. However, you are free to use numbers outside of this range as well.

Works exactly the same as sprite.SetAnchor (see Sprites & Animation).


StaticText.xscale StaticText.yscale Allows you to stretch and squish static text objects, similarly to sprite objects.


StaticText.Scale( xscale, yscale) Same as setting StaticText.xscale and StaticText.yscale at the same time.


StaticText.MoveBelow(staticText otherStaticTextObject) StaticText.MoveBelow(text textObject) Moves a static text object below another static text object or text object.
They must both be on the same layer.


StaticText.MoveAbove(staticText otherStaticTextObject) StaticText.MoveAbove(text textObject) Moves a static text object above another static text object or text object.
They must both be on the same layer.


StaticText.text (Read-Only) Gets the text in the StaticText Object.


StaticText.SetText( text) Sets the text in the StaticText Object.
If the static text object is inactive when this is called, the object will reactive itself.


StaticText.SetFont( fontName) Sets the font of the text. It's the only way of changing text's font.
Returns an error if the font doesn't exist.

* Note that the font is not changed until call StaticText.SetText() unless calling AutoRefreshStaticText(true).


StaticText.SetCharSpacing( space = nil) Sets the number of pixels that separate characters on the same line in the static text object.
Negative = less space between characters, Positive = more space between characters.

The default value is 3.
Fonts can specify a <charspacing> tag that sets this value.

Assign nil to reset this value to the value provided by the font (or the default value of 3).

* Note that the charspacing is not changed until call StaticText.SetText() unless calling AutoRefreshStaticText(true).


StaticText.SetLineSpacing( line = 0) Number of pixels that separate different lines in the static text object. The number you enter here gets added on to the active font's regular line spacing.
Negative = less space between lines, Positive = more space between lines.


By default, the amount of pixels between lines is 1.5 * the height of the font's "space" character.
If the font has a <linespacing> tag, its number value will be the amount of pixels between lines instead.

Assign nil to reset the line spacing to its original value.

* Note that the linespacing is not changed until call StaticText.SetText() unless calling AutoRefreshStaticText(true).


StaticText.Move( x, y) Moves the text to a new position, relative to the current static text object's position.


StaticText.MoveTo( x, y) Moves the text to a new position, relative to the static text object's parent's position.


StaticText.MoveToAbs( x, y) Moves the text to a new position, relative to the bottom left corner of the window.


StaticText.width (Read-only) Shortcut to StaticText.GetTextWidth().


StaticText.height (Read-only) Shortcut to StaticText.GetTextHeight().


StaticText.GetTextWidth() Returns the width of the currently set text in pixels.
Will give the same result even if the text isn't finished typing.

Also ignores StaticText.xscale.

Useful if you want to center your text.


StaticText.GetTextHeight() Returns the height of the curently set text in pixels.
Will give the same result even if the text isn't finished typing.

Also ignores StaticText.yscale.

Note: This function is not 100% accurate.
