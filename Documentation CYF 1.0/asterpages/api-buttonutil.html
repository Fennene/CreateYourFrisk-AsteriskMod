The ButtonUtil object [E/M/W]



You can use this object to modify buttons.



Button ButtonUtil.FIGHT This function returns property of FIGHT Button as Button object.


Button ButtonUtil.ACT This function returns property of ACT Button as Button object.


Button ButtonUtil.ITEM This function returns property of ITEM Button as Button object.


Button ButtonUtil.MERCY This function returns property of MERCY Button as Button object.


ButtonUtil.SetActives( fight = true, act = true, item = true, mercy = true) Sets the activates of all buttons at once.

  • fight: active of FIGHT Button.
  • act: active of ACT Button.
  • item: active of ITEM Button.
  • mercy: active of MERCY Button.

* Note: you can NOT inactive all buttons.


The Button object



You can modify position, activate and texture of buttons.



button.x Horizontal position of sprite relative to the normal positon.


button.y Vertical position of sprite relative to the normal positon.


button.Move( x, y) Moves the sprite x pixels right, and y pixels up.

See button.x and button.y.


button.MoveTo( x, y) Same as setting x and y simultaneously.

See button.x and button.y.


button.ResetPosition() Sets the button's position to normal position.
This is same as button.MoveTo(0, 0).


button.SetActive( active) Either true or false. If false, the button disappear and the player can't select that.


button.GetActive() This function returns activate of the button.


button.SetSprite( inactiveTexturePath, activeTexturePath) Change the button's current texture.

  • inactiveTexturePath: the image while the player doesn't select.
  • activeTexturePath: the image when the player select.


button.SetColor(r, g, b, a = 1.0) Sets the button's color, as 3 or 4 values from 0 to 1.
For example, button.SetColor(1.0, 0.0, 0.0) makes the button red.
Black areas are not affected by coloration.
The 4th value is the alpha (transparency) value of the button.


button.SetColor32(r, g, b, a = 255) Sets the button's color, as 3 or 4 integer values from 0 to 255.
For example, button.SetColor32(255, 0, 0) makes the button red.
Black areas are not affected by coloration.
The 4th value is the alpha (transparency) value of the button.


button.ResetColor() Resets the button's color.
This is same as button.SetColor32(1, 1, 1, 1) or button.SetColor32(255, 255, 255, 255).
