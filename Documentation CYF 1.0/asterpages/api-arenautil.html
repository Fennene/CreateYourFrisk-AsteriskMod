The ArenaUtil object [E/M/W]


namespace: AsteriskMod
code: AsteriskMod/Arena/ArenaUtil.cs


This is for changing arena's color and parameters of arena's text.



ArenaUtil.centerabsx (Read-only) Returns horizontal position of arena relative to the bottom left corner of the screen.


ArenaUtil.centerabsy (Read-only) Returns vertical position of arena relative to the bottom left corner of the screen.


ArenaUtil.isMoving (Read-only) This is same to Arena.isMoving.


ArenaUtil.isResizing (Read-only) This is same to Arena.isResizing.


ArenaUtil.isModifying (Read-only) This is same to Arena.isModifying.


ArenaUtil.offsetx Horizontal position of arena relative to initial position.
This is useful on states that arena can't be moved by script like ACTIONSELECT.


ArenaUtil.offsety Vertical position of arena relative to initial position.
This is useful on states that arena can't be moved by script like ACTIONSELECT.


ArenaUtil.SetOffset( x, y) Same as setting ArenaUtil.offsetx and ArenaUtil.offsety simultaneously.


ArenaUtil.relativewidth Width of arena relative to initial size.
This is useful on states that arena can't be moved by script like ACTIONSELECT.
This parameter does NOT reflecte to arenasize variable.

You can not set the size of arena to negative by using this.


ArenaUtil.relativeheight Height of arena relative to initial size.
This is useful on states that arena can't be moved by script like ACTIONSELECT.
This parameter does NOT reflecte to arenasize variable.

You can not set the size of arena to negative by using this.


ArenaUtil.SetRelativeSize( width, height, immediate = false) Same as setting ArenaUtil.relativewidth and ArenaUtil.relativeheight simultaneously.


ArenaUtil.SetBorderColor( r, g, b, a = 1) Sets the border's color of arena, as 3 or 4 values from 0 to 1.
For example, ArenaUtil.SetBorderColor(1.0, 0.0, 0.0) makes the text red.
Black areas are not affected by coloration.
The 4th value is the alpha (transparency) value of the border.


ArenaUtil.SetBorderColor32( r, g, b, a = 255) Sets the border's color of arena, as 3 or 4 integer values from 0 to 255.
For example, ArenaUtil.SetBorderColor32(255, 0, 0) makes the text red.
Black areas are not affected by coloration.
The 4th value is the alpha (transparency) value of the border.


shader ArenaUtil.bordershader The shader object linked to arena's border.

It can't probably use Wave shader.


ArenaUtil.SetInnerColor( r, g, b, a = 1) Sets the inner color of arena, as 3 or 4 values from 0 to 1.
For example, ArenaUtil.SetInnerColor(1.0, 0.0, 0.0) makes the text red.
Black areas are not affected by coloration.
The 4th value is the alpha (transparency) value of that.


ArenaUtil.SetInnerColor32( r, g, b, a = 255) Sets the inner color of arena, as 3 or 4 integer values from 0 to 255.
For example, ArenaUtil.SetInnerColor32(255, 0, 0) makes the text red.
Black areas are not affected by coloration.
The 4th value is the alpha (transparency) value of that.


shader ArenaUtil.innershader The shader object linked to arena's inner.

It can't probably use Wave shader.


ArenaUtil.dialogtextx Horizontal position of the text relative to the initial position.

If the object of the text in dialog box is not initialized, the position is not changed.


ArenaUtil.dialogtexty Vertical position of the text relative to the initial position.

If the object of the text in dialog box is not initialized, the position is not changed.


ArenaUtil.DialogTextMove( x, y) Moves the text in dialog box to the position relative to current position.

If the object of the text in dialog box is not initialized, the position is not changed.


ArenaUtil.DialogTextMoveTo( x, y) Moves the text in dialog box to the position relative to initial position.
This is same to set ArenaUtil.dialogtextx and ArenaUtil.dialogtexty at the same time.

If the object of the text in dialog box is not initialized, the position is not changed.


sprite ArenaUtil.GetDialogTextLetters() Returns a table containing sprite objects representing every letter to be displayed in the text object's current line of text.

If the text in dialog box is not initialized or does not shows, returns empty table.

See text.GetLetters() for more info.


ArenaUtil.AsteriskChar Gets or sets character of asterisk (*). It is useful when you use fonts in other languages (for example, Japanese).
I don't recommend to change asterisk frequently.


ArenaUtil.SetFullWidthAsterisk( active) When line breaks as \r, coordinates (expands) the space.
This is useful if Asteirsk is sets full-width by ArenaUtil.AsteriskChar.

This is useful to translate.
日本語訳や詳細は Japanese Localization [日本語対応]を参照してください。


ArenaUtil.SetAutoFontCommandActive( active) This is useful to translate.
日本語訳や詳細は Japanese Localization [日本語対応]を参照してください。


ArenaUtil.SetAutoFontCommandFont( fontName) This is useful to translate.
日本語訳や詳細は Japanese Localization [日本語対応]を参照してください。


ArenaUtil.SetDialogTextVolume( value) Sets the dialog text's volume of sound(voice). value should be between 0.0 (muted) and 1.0 (full volume). This is 1.0 by default.

If the text in dialog box is not initialized, nothing happens.


ArenaUtil.GetDialogTextVolume() Returns the dialog text's volume of sound(voice).

If the text in dialog box is not initialized, return 1.


ArenaUtil.playeroffsetx Horizontal position of the player relative to initial position on selection ("on selection" means for example ENEMYSELECT, ACTMENU and MERCYMENU).


ArenaUtil.playeroffsety Vertical position of the player relative to initial position on selection ("on selection" means for example ENEMYSELECT, ACTMENU and MERCYMENU).


ArenaUtil.SetPlayerOffsetOnSelection( x, y) Same as setting ArenaUtil.playeroffsetx and ArenaUtil.playeroffsety simultaneously.
