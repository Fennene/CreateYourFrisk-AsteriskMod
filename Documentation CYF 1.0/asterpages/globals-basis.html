GlobalScript



GlobalScript is one of very strong feature in Asterisk Mod.
Asterisk Mod adds new types of scripts and files, named GlobalsScripts Script and Enum Files.


GlobalsScripts Script



Basic Setup


The GlobalsScripts scripts are located in YOUR MOD/Lua/Globals/Scripts.
You can use this script as storing value with variable and as library containing general functions.
Variables and functions of this scripts are same values anywhere unlike require.

You should use the variable Globals when you access GlobalsScripts scripts. You don't need to define many variables and functions in Encounter Script anymore!



Globals variable (The Globals Objects) [E/M/W/S/GS]

namespace: AsteriskMod.GlobalScript
code: AsteriskMod/GlobalScript/GlobalsScripts.cs


This variable can also use in GlobalsScripts scripts, but an error will have occurred if you access that script itself from the script.


Globals.Exists( scriptName) Returns whether specified script exists or not.


Globals.GetScriptList() Returns a list of loaded scripts.


script Globals[ scriptName]
or script Globals.GetScript( scriptName) Returns specified script as a script object.
If you specify the script that doesn't exist, an error will have occurred.


object Globals.GetVar( scriptName, variable_name)
or object Globals.Get( scriptName, variable_name) This is same as script.GetVar().


Globals.SetVar( scriptName, variable_name, value)
or Globals.Set( scriptName, variable_name, value) This is same as script.SetVar().


Globals.Call( scriptName, function_name, params any types arguments) This is same as script.Call().
The argument arguments is params, so there is no problem even if the number of argumant's element is 0.


Script(Table) Globals.GetRawScript( scriptName) Returns script as like require.



Available variables, function and objects


You can use almost features written [E/M/W] or [E/M/W/S] in this document to the scripts.


Enum Files




Basic Setup


The Enum Files are located in YOUR MOD/Lua/Globals/Enums.
The Engine creates enum dynamically from those files.
The extension is .enum.

You should use the variable Enums when you access the enums.



Enums variable (The Enums Objects) [E/M/W/S/GS]

namespace: AsteriskMod.GlobalScript
code: AsteriskMod/GlobalScript/DynamicEnums.cs


Should assign path relative Lua/Globals/Enums without extension to fileName.


Enums.Exists( fileName) Returns whether specified file exists or not.


enum Enums.Get( fileName)or enum Enums[ fileName] Gets specified enum.



The format & Example


The format have been made based on C#'s enum.
.enum file format 

// this is comment out.
// Test.enum
A,
B,
C,
D,
E
-> Enums["Test"].A ~ Enums["Test"].E
