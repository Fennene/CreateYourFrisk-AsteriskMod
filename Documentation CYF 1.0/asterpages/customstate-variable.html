Available Variables, Functions and Objects in CustomState



There is a list of variables, functions, and objects you can use in State Script.
You can use almost features written [E/M/W] in this document in state scripts.

If you want to know each feature, see the other pages.

Variables

  • isCYF: whether is CYF or not. always true.

  • isModifiedCYF: whether is modified CYF or original CYF. always true.

  • Asterisk: whether AsteriskMod is enabled or disabled. always true.

  • AsteriskExperiment: whether Experimental Features is enabled or disabled.

  • safe: whether CYF's safe mode is enable or disable.

  • windows: whether the client uses windows OS or not.

  • CYFversion: a value shows CYF's version. always v0.6.5.

  • AsteriskVersion: a value shows AsteriskMod's version.

  • script Encounter: returns Encounter Script as script object.


Functions

  • SetGlobal: sets a value that can access anywhere in script.

  • object GetGlobal: gets a value set by SetGlobal.

  • SetRealGlobal: sets a value that can access anywhere in all mods until CYF closes.

  • object GetRealGlobal: gets a value set by SetRealGlobal.

  • SetAlMightyGlobal: sets a value and save that to CYF's save data.

  • object GetAlMightyGlobal: gets a value from CYF's save data.

  • DEBUG: opens debugger and shows a data from a value inputted.

  • EnableDebugger: sets whether debugger is enabled or disabled.

  • State: changes mod's state.

  • GetCurrentState: gets current mod's state.

  • SetFrameBasedMovement: changes the movement's base of the player's soul.

  • SetAction: moves the player's soul on any button.

  • AllowPlayerDef: whether include the player's def or not to calculate the damage of player.

  • BattleDialog or BattleDialogue: sets text in dialog box and go to the state DIALOGRESULT.

  • CreateLayer: creates layer. returns whether success or not.

  • LayerExists: returns whether layer exsits or not.

  • sprite CreateSprite: creates sprite object.

  • CreateProjectileLayer: IDK. See the section Projectile Managemnt.

  • text CreateText: creates text object.

  • bullet CreateProjectile and CreateProjectileAbs: creates bullet object.

  • SetPPCollision: sets whether use PPCollisionSystem or not to check collide the player's soul and bullet.

  • Audio: the Audio object.

  • NewAudio: the NewAudio object.

  • ArenaUtil: the ArenaUtil object.
