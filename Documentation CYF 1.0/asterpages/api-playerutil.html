The PlayerUtil object [E/M/W]



You can use this object to modify the player's UI.


Note: You shouldn't call below methods in EncounterStarting().
You should call them in FirstFrameUpdate() instead of it.

(See Game events.)


The Player's All UI

The Player's All UI contains the player's name, LV, label of HP, bars of HP and text of HP.


PlayerUtil.UIMove( x, y) Moves the player's UI on its last position.


PlayerUtil.UIMoveTo( x, y) Moves the player's UI to this position relative to normal position.


PlayerUtil.ResetUIPosition() Moves the player's UI on its last position. This is same as PlayerUtil.UIMoveTo(0, 0).


PlayerUtil.GetUIPosition() returns Obtain the position of the player's UI relative to normal position.


The Player's NameUI

The Player's NameUI contains the player's name and LV.


PlayerUtil.NameUIMove( x, y) Moves the player's NameUI on its last position.


PlayerUtil.NameUIMoveTo( x, y) Moves the player's NameUI to this position relative to normal position.


PlayerUtil.ResetNameUIPosition() Moves the player's NameUI on its last position. This is same as PlayerUtil.NameUIMoveTo(0, 0).


PlayerUtil.GetNameUIPosition() returns Obtain the position of the player's NameUI relative to normal position.


PlayerUtil.SetLV( lv) Set the player's LV as string.
This means you can set the LV other than numbers.


PlayerUtil.SetNameColor( r, g, b, a = 1) Sets the text's color of the player's name, as 3 or 4 values from 0 to 1.
For example, PlayerUtil.SetNameColor(1.0, 0.0, 0.0) makes the text red.
Black areas are not affected by coloration.
The 4th value is the alpha (transparency) value of the text.


PlayerUtil.SetNameColor32( r, g, b, a = 255) Sets the text's color of the player's name, as 3 or 4 integer values from 0 to 255.
For example, PlayerUtil.SetNameColor32(255, 0, 0) makes the text red.
Black areas are not affected by coloration.
The 4th value is the alpha (transparency) value of the text.


PlayerUtil.SetLVColor( r, g, b, a = 1) Sets the text's color of the player's LV, as 3 or 4 values from 0 to 1.


PlayerUtil.SetLVColor32( r, g, b, a = 255) Sets the text's color of the player's LV, as 3 or 4 integer values from 0 to 255.


The Player's HPUI

The Player's HPUI contains the player's label of HP, bars of HP and text of HP.


PlayerUtil.HPUIMove( x, y) Moves the player's HPUI on its last position.


PlayerUtil.HPUIMoveTo( x, y) Moves the player's HPUI to this position relative to normal position.


PlayerUtil.ResetHPUIPosition() Moves the player's HPUI on its last position. This is same as PlayerUtil.HPUIMoveTo(0, 0).


PlayerUtil.GetHPUIPosition() returns Obtain the position of the player's HPUI relative to normal position.


PlayerUtil.SetHPControlOverride( active) Either true or false. If true, bars and texts showing HP do NOT update anymore.
This is useful for using PlayerUtil.SetHP(), PlayerUtil.SetHPBarLength() and PlayerUtil.SetHPText().


The Player's Label of HP

The Player's Label of HP is the sprite written "HP". The sprite itself can be changed by swapping texture in Sprites/UI/spr_hpname_0 and Sprites/UI/Battle/spr_hpname_0


PlayerUtil.SetHPLabelColor( r, g, b, a = 1) Sets the sprite's color of HP's label, as 3 or 4 values from 0 to 1.
For example, PlayerUtil.SetHPLabelColor(1.0, 0.0, 0.0) makes the sprite red.
Black areas are not affected by coloration.
The 4th value is the alpha (transparency) value of the sprite.


PlayerUtil.SetHPLabelColor32( r, g, b, a = 255) Sets the sprite's color of HP's label, as 3 or 4 integer values from 0 to 255.
For example, PlayerUtil.SetHPLabelColor32(255, 0, 0) makes the sprite red.
Black areas are not affected by coloration.
The 4th value is the alpha (transparency) value of the sprite.


The Player's Bars of HP

The Player's Bars of HP is the bar displaying the player's HP.
HPBarBG(Background Bar) is the red(normally) bar shows the player's total HP. HPBarBGFill(Fill Bar) is the yellow(normally) bar shows the player's current HP.


PlayerUtil.SetHP( newHP, newMaxHP, updateHPText = false) Sets the length of fill bar, as 2 integer values.

  • updateHPText: If true, sets the text showing HP as based on 2 integer values.
This function does NOT change the length of background bar and max fill bar.


PlayerUtil.SetHPBarLength( newMaxHP) Sets the length of background bar, as integer values.


PlayerUtil.SetHPBarBGColor( r, g, b, a = 1) Sets the color of background bar, as 3 or 4 values from 0 to 1.
For example, PlayerUtil.SetHPBarBGColor(0.0, 0.0, 1.0) makes the bar blue.
Black areas are not affected by coloration.
The 4th value is the alpha (transparency) value of the bar.


PlayerUtil.SetHPBarBGColor32( r, g, b, a = 255) Sets the color of background bar, as 3 or 4 integer values from 0 to 255.
For example, PlayerUtil.SetHPBarBGColor32(0, 0, 255) makes the bar blue.
Black areas are not affected by coloration.
The 4th value is the alpha (transparency) value of the bar.


PlayerUtil.SetHPBarFillColor( r, g, b, a = 1) Sets the color of fill bar, as 3 or 4 values from 0 to 1.


PlayerUtil.SetHPBarFillColor32( r, g, b, a = 255) Sets the color of fill bar, as 3 or 4 integer values from 0 to 255.


The Player's Text of HP


PlayerUtil.HPTextMove( x, y) Moves the player's text of HP on its last position.


PlayerUtil.HPTextMoveTo( x, y) Moves the player's text of HP to this position relative to normal position.


PlayerUtil.ResetHPTextPosition() Moves the player's text of HP on its last position. This is same as PlayerUtil.HPUIMoveTo(0, 0).


PlayerUtil.GetHPTextPosition() returns Obtain the position of the player's text of HP relative to normal position.


PlayerUtil.SetHPText( hpText) Set the player's HP as string.


PlayerUtil.SetHPTextColor( r, g, b, a = 1) Sets the text's color of the player's HP, as 3 or 4 values from 0 to 1.
For example, PlayerUtil.SetHPTextColor(1.0, 0.0, 0.0) makes the text red.
Black areas are not affected by coloration.
The 4th value is the alpha (transparency) value of the text.


PlayerUtil.SetHPTextColor32( r, g, b, a = 255) Sets the text's color of the player's HP, as 3 or 4 integer values from 0 to 255.
For example, PlayerUtil.SetHPTextColor32(255, 0, 0) makes the text red.
Black areas are not affected by coloration.
The 4th value is the alpha (transparency) value of the text.
