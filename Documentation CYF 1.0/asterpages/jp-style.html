JPStyle オブジェクト [E/M/W/S]



このオブジェクト(クラス)は 日本語対応をスムーズに行うためのものです。


JPStyle.SetActive( active) 以下にある機能を一括で有効または無効にします。


プレイヤー名関係


JPStyle.JPName プレイヤー名が日本語対応している(フォントがjpnameである)かどうか取得します。
代入(設定)は、JPStyle.SetJapaneseNameActive()を設定することと同等です。

デフォルト値はCYFの言語設定に依存します。


JPStyle.SetJPName( active) JPStyle.SetJapaneseNameActive()のショートカット用。


JPStyle.SetJapaneseNameActive( active) プレイヤー名を日本語対応させます。
これは、 

-- "active"がtrueの場合
PlayerUtil.Name.SetFont("jpname")
PlayerUtil.Name.Move(0, 6) -- 位置調整
-- "active"がfalseの場合
PlayerUtil.Name.SetFont("uibattlesmall")
PlayerUtil.Name.Move(0, -6) -- 位置調整
とほぼ同等です。

既にtrueであるのに、trueを代入した場合、または既にfalseであるのに、falseを代入した場合は、何も起こりません。


フォント関係


JPStyle.AutoFontCoordinating アスタリスクを伴わない改行で文字のズレを直すかどうかを返します。
詳細はJPStyle.SetAutoFontCoordinatingActive()を参照してください。

デフォルトではfalse。


JPStyle.SetAutoFontCoordinatingActive( active) 公式日本語版では、アスタリスク(*のこと)は全角のアスタリスクを使用します。
ただ単に、日本語フォント(デフォルトではjp)に変えてArenaUtil.AsteriskChar = "＊"を実行するだけでは、
アスタリスクを伴わない改行(\rによる改行)で、文字がずれます。
この関数はその文字のズレを調整します。

基本的に、jpフォント用です。

これは、 

-- "active"がtrueの場合
ArenaUtil.AsteriskChar = "＊"
ArenaUtil.DialogTextMove(0, -5) -- 位置調整
-- 加えてCYFエンジン側で改行の内部調整を実行するようになる。
-- "active"がfalseの場合
ArenaUtil.AsteriskChar = "*"
ArenaUtil.DialogTextMove(0, 5) -- 位置調整
とほぼ同等です。

既にtrueであるのに、trueを代入した場合、または既にfalseであるのに、falseを代入した場合は、何も起こりません。


















AsteriskModにはCYFエンジン及びModを、日本語化するための機能がいくつか追加されています。


CYFエンジン自体の日本語化は、Mod選択画面の右上にある「Options」を選択し、(オプション選択用のウインドウ的なものが開くので) 「Asterisk Mod Option」を選択すると、言語を変える設定のボタン(「Language:English」と表示されていると思います)をクリックしてください。


エンジンが日本語であると、Mod内でプレイヤー名が日本語で表示されます。(Mod製作者が何かしら設定していない限り。)


Modの日本語対応



Modを日本語化したいのであれば、まず EncounterStarting()に JPStyle.SetActive(Language == "Japanese") と書くことをおすすめします。 

function EncounterStarting()
    JPStyle.SetActive(Language == "Japanese")
end

JPStyle.SetActive() は日本語化の機能を全て有効にするので、各機能の詳細や一部の機能のみを有効化したい場合は、JPStyleオブジェクトセクションを参照してください。



AsteriskModには、Defaultフォルダに、日本語のフォントが既に入っています。
使用できるフォントは以下の通りです。

  • jp - 公式日本語版の汎用フォントです。
  • jp2 - 戦闘中のモンスターのセリフ用のフォントです。実際の公式日本語版のフォントではないです。
  • jpname - 公式日本語版の戦闘時のプレイヤー名表示用フォントです。

対応している文字は、半角英数字記号と全角ひらがなと全角カタカナです。
jpとjp2はそれに加えて、全角英数字記号を含みます。

使用方法としては テキストコマンドの[font:jp]や、TextオブジェクトのText.SetFont("jp")ですが、
JPStyleオブジェクトを使用すれば、テキストコマンドを書く手間が減ります。( 詳細は(ry )



また、AsteriskModには、Defaultフォルダに、日本語のボタンも既に入っています。
日本語版のボタンのテクスチャのファイル名は以下の通り。(拡張子はpngです。)

各ボタン名 非アクティブ時 アクティブ時
たたかう fightbt_2 fightbt_3
こうどう actbt_2 actbt_3
アイテム itembt_2 itembt_3
みのがす mercybt_2 mercybt_3
