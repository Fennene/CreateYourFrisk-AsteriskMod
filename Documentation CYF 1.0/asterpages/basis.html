Asterisk Mod is a folk of Create Your Frisk v0.6.5.
It adds features to customize the UIs of CYF, and you can write more succinct and simple codes.

The Purpose of Asterisk Mod



AsteriskMod is developed focus on customizability of UIs, ease to write lua codes, and ease to create overworld in a mod.
Based on CYF v0.6.5, it has added features I have thought I want, as much as I can have modified.
It provides the objects that related each UIs are put themselves together into one object. For example, it define UIs displaying the player's status as "The PlayerUtil Object".
Excepted some features, it does not modify original (CYF's) objects basically.

More info about each features, see "Features overview"; If you wanna tell me "This is useless beacuse there is CYFv0.6.6.", see "Relation with v0.6.6 and Compatibility with v0.6.6".

Features Overview



AsteriskMod adds and modifies below features to CYF:

  • Copying Error: You can copy error message.

  • Mod's Info: modifies mod's information and working.
  • Code's Style: modifies script's working.

  • Special Variables: provides variables: checking whether modified CYF or not, modifying the text of MERCYMENU, changing states, modifying the bar's color of enemy's HP, and controlling FightUI.

  • Game Events: adds functions which run some the time point. (but these are .)

  • Misc. functions: provides functions: returning current action, going to CustomState, and getting horizontal position when shake in progress.

  • The Misc Object: Tentatively I made this, but I had forgotten to exist shader...

  • Projectile management: adds variables which moves bullet automatically.

  • Sprites and Animation: adds function which checks layer exists.

  • The Text Object: Text.Move()!!1!!!!!1!!!!!!

Asterisk Mod has below features:

  • Engine: controls the engine itself.

  • AppData: opens AppData directory.

  • PlayerUtil: controls the UIs displaying the player's status, changes default damage and invincible of the player, changes the sprite and animation of Fight UI, and gets enemy's damage with some arguments.

  • ButtonUtil: controls the buttons.

  • ArenaUtil: changes default position and size of arena, changes the color of arena, changes the position of dialog text, gets sprites of dialog text object, and changes the position of player on selection.

  • Japanese Localization: CYFの日本語対応！？

  • GlobalsScript Script & Globals: Global's script.

  • .enum file & Enums: Dynamic enum.

  • CustomState: You can create your own state with state scripts and The StateEditor Object!!1!!!!!1!!!!!!

See each sections for more information.
"Asterisk Mod Features" Mods contain some examples.

* Note that AsteriskMod does NOT support Overworld!!
