Asterisk Mod



Asterisk Mod v0.5.2.5 by Nil256 [FenneneProject]

Asterisk Mod is a folk of original Create Your Frisk. That adds some strong functions and objects.



Asterisk Mod adds and modifies below features to original CYF:

  • PlayerUtil: controls the UI of the player's status.

  • ButtonUtil: controls the Buttons.

  • Special Variables: control the UI for example, text of MERCYMENU and enemy's life bar.

  • Game Events: adds functions which run some the time point.

  • The Misc Object: reverses and rotates screen.


There are new 3 symbols in this documentaion and those means:
  • are only usable in AsteriskMod.

  • are only usable in AsteriskMod when "Experimental Features" is enabled.

  • Indicates content new or changed CYF's content in Asterisk Mod.


Asterisk Mod also adds some options.
You can see that info and options when you select Asterisk Mod Option.
There are below options available:

  • Experimental Features: Allow experimental features.
    Experiment features contains the contents which will be added future update and the contents that may occur some error or may be unstable.

I may be add Language Option (Japanese) in future update...?



You can find string Asterisk Mod Modification in source codes.
That is codes which modified or added by *Mod.
