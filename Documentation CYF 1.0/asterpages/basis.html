Asterisk Mod is a folk of original Create Your Frisk.
It adds features to customize the UIs of CYF and you can write more succinct and simple codes.

The Purpose of Asterisk Mod



Asterisk Mod is created to focus on controlling CYF's original UIs.
Related each UIs are put themselves together into a object. For example, Controlling UIs related to display the Player's status is named "PlayerUtil".
Yeah, the names' suffix of the most objects is "Util".
That's because I don't know about more appropriate words or suffix in English, but you can know that objects are added by Asterisk Mod and means which UIs, just by seeing the object's name.
However, that is not exactly wrong.
If you use this mod, you may not need to create Fake UI and the engine may works quicker.
The main purpose that Asterisk Mod is made is for creating more easier what you come up and your good ideas.


Features



Asterisk Mod adds below features to CYF:

  • Copying Error: You can copy error message.

  • Mod's Description: adds your mod's description.

  • Special Variables: modifies the text of MERCYMENU and the color of enemies' life bar.

  • Game Events: adds functions which run some the time point.

  • Misc. functions: has function that returns current action.

  • The Misc Object: reverses and rotates camera.

  • Sprites and Animation: checks layer exists.

Asterisk Mod has below features:

  • PlayerUtil: controls the UI of the player's status.

  • ButtonUtil: controls the buttons.

  • ArenaUtil: controls the object in arena and itself.

  • CustomState: You can create your own state.

See each sections for more information.
"Asterisk Mod Features" Mod contains some examples.
