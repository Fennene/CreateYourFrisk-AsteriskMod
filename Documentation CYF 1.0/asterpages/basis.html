* Asterisk Mod



Asterisk Mod v0.5.2.9 by Nil256 [FenneneProject]

Asterisk Mod is a folk of original Create Your Frisk. That adds some strong functions and objects.



Nil256 (AsteriskMod's developer) is not good at English. There may be some mistakes of English in this documentation, Sorry.



Asterisk Mod adds and modifies below features to original CYF:

  • Copying Error: You can copy error message.

  • Mod's Info: You can add your mod's description.

  • PlayerUtil: controls the UI of the player's status.

  • ButtonUtil: controls the Buttons.

  • ArenaUtil: controls the object in arena.

  • CustomState: You can create your own state.

  • Special Variables: controls the UI for example, text of MERCYMENU and enemy's life bar.

  • Game Events: adds functions which run some the time point.

  • The Misc Object: reverses and rotates screen and shows message box.

  • The Text Object: sets voice's volume.


There are new 5 symbols in this documentaion and those means:
  • are only usable in AsteriskMod.

  • are only usable in AsteriskMod when "Experimental Features" is enabled.

  • means it may be deleted by Asterisk Mod v0.5.4.
  • Indicates new or changed CYF's content in Asterisk Mod.
  • Indicates new or changed content in Asterisk Mod v0.5.3(v0.5.2.9).


You can find string Asterisk Mod Modification in source codes.
That is codes which modified or added by Asterisk Mod.
